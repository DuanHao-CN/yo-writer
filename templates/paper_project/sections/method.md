# 四、研究设计

## （一）样本选取与数据来源

本文以2011—2022年中国A股上市公司为研究样本。选择2011年作为起始年份，主要基于北京大学数字普惠金融指数的数据起点。在样本筛选过程中，本文按照以下标准进行处理：（1）剔除金融行业上市公司，因其资产负债结构和经营模式与一般企业差异较大；（2）剔除ST、*ST类上市公司，以排除财务异常对结果的干扰；（3）剔除关键变量存在严重缺失的样本。经过上述处理，最终获得约26000—30000个企业—年度观测值的非平衡面板数据。

本文的数据来源如下：企业专利数据来自中国研究数据服务平台（CNRDS），该数据库提供了上市公司按专利类型（发明专利、实用新型专利和外观设计专利）分类的详细申请和授权信息（黎文靖和郑曼妮，2016）；城市层面的数字经济发展水平以北京大学数字金融研究中心编制的数字普惠金融指数衡量（谢绚丽等，2018）；上市公司财务数据和公司治理数据来自国泰安（CSMAR）数据库；城市层面的经济社会数据来自《中国城市统计年鉴》；工具变量所需的历史邮电数据来自《中国统计年鉴》历史卷。为消除极端值的影响，本文对所有连续变量在1%和99%分位数处进行了缩尾处理（Winsorize）。

## （二）变量定义

### 1. 被解释变量

本文的被解释变量为企业创新质量，通过区分实质性创新与策略性创新两个维度进行衡量。

**实质性创新（SubInnov）**：以企业当年发明专利申请数量加1后取自然对数衡量，即SubInnov_it = ln(发明专利申请量_it + 1)。发明专利需要经过严格的实质性审查，具有较高的技术含量和创新水平，能够较好地反映企业的实质性技术突破能力。

**策略性创新（StrInnov）**：以企业当年实用新型专利申请数量加1后取自然对数衡量，即StrInnov_it = ln(实用新型专利申请量_it + 1)。实用新型专利仅需形式审查，技术门槛较低，部分企业可能出于迎合创新补贴政策等策略性动机申请此类专利。

在稳健性检验中，本文还使用专利授权数量替换申请数量进行重新估计。

### 2. 核心解释变量

**数字经济发展水平（DigEcon）**：采用北京大学数字普惠金融指数的城市级总指数衡量。该指数基于蚂蚁集团的交易账户大数据编制，涵盖覆盖广度、使用深度和数字化程度三个维度，是目前学术界衡量中国城市数字经济发展水平最具权威性和认可度的指标之一（郭峰等，2020）。本文将企业按注册地所在城市与数字普惠金融指数进行匹配，并对指数取自然对数以降低量纲差异。在稳健性检验中，本文还采用赵涛等（2020）构建的城市数字经济综合发展指数作为替代指标。

### 3. 控制变量

为尽可能减少遗漏变量偏误，本文在企业层面和城市层面分别控制了以下变量：

**企业层面控制变量**：（1）企业规模（Size），以总资产的自然对数衡量；（2）企业年龄（Age），以企业成立年限的自然对数衡量；（3）资产负债率（Lev），以总负债与总资产之比衡量；（4）资产收益率（ROA），以净利润与总资产之比衡量；（5）营业收入增长率（Growth），衡量企业成长性；（6）经营现金流（CF），以经营活动现金流净额与总资产之比衡量；（7）研发投入强度（RD），以研发支出与营业收入之比衡量；（8）股权集中度（Top1），以第一大股东持股比例衡量；（9）董事会规模（Board），以董事会人数的自然对数衡量。

**城市层面控制变量**：（1）经济发展水平（GDP_pc），以城市人均GDP的自然对数衡量；（2）政府科技投入（GovTech），以地方财政科技支出占财政总支出的比例衡量；（3）金融发展水平（Finance），以年末金融机构贷款余额与GDP之比衡量；（4）人力资本水平（HC），以每万人在校大学生数衡量。

### 4. 机制变量

**知识溢出（KnowSpill）**：采用城市技术市场成交额的自然对数衡量（ln(技术市场成交额 + 1)），该指标反映了一个城市技术知识的交易和流动活跃程度，是衡量知识溢出强度的常用代理变量。

**资源配置效率**：从两个角度进行衡量。一是企业全要素生产率（TFP），采用Levinsohn和Petrin（2003）的LP方法进行估计；二是融资约束指数（SA），参考Hadlock和Pierce（2010）的方法，以企业规模和企业年龄构建SA指数（取绝对值，数值越大表示融资约束越严重）。

## （三）模型设定

### 1. 基准回归模型

为检验数字经济对企业创新质量的影响，本文构建如下双向固定效应模型：

$$Innovation_{it} = \alpha + \beta \cdot DigEcon_{ct} + \gamma \cdot Controls_{it} + \mu_i + \lambda_t + \varepsilon_{it} \quad (1)$$

其中，$Innovation_{it}$分别以$SubInnov_{it}$（实质性创新）和$StrInnov_{it}$（策略性创新）代入；$DigEcon_{ct}$为企业$i$所在城市$c$在$t$年的数字经济发展水平；$Controls_{it}$为控制变量向量；$\mu_i$为企业固定效应，用于控制不随时间变化的企业层面不可观测因素；$\lambda_t$为年份固定效应，用于控制宏观经济环境的时间趋势；$\varepsilon_{it}$为随机扰动项。系数$\beta$是本文关注的核心参数，若$\beta$在实质性创新回归中显著为正，而在策略性创新回归中不显著或显著更小，则支持假设H1和H1a。标准误在城市层面进行聚类调整，以修正同一城市内企业之间误差项的潜在相关性。

### 2. 内生性处理

数字经济与企业创新之间可能存在内生性问题：一方面，创新活跃的地区可能对数字基础设施有更大的需求，导致反向因果；另一方面，地区制度环境、文化氛围等不可观测因素可能同时影响数字经济发展和企业创新，造成遗漏变量偏误。为此，本文采用两种策略进行因果识别：

**策略一：工具变量法（IV-2SLS）**。本文参考黄群慧等（2019）的做法，以1984年各城市每百万人邮电局数量与上一年全国互联网用户数的交互项作为工具变量。**相关性论证**：历史邮电基础设施是当代数字基础设施的物理前身——光纤网络沿既有电话线路铺设，移动通信基站依托原有通信节点布局——因此历史邮电局密度与当前数字经济发展水平高度相关。**外生性论证**：1984年的邮电局布局主要受计划经济时期行政决策的影响，反映的是当时的通信服务行政规划而非市场力量，与2011—2022年企业的创新活动不存在直接因果关系。为增强排除限制的可信度，本文在第一阶段回归中额外控制了城市1984年人口规模（对数），以排除历史邮电局布局通过城市历史规模影响当前创新环境的间接渠道（Goldsmith-Pinkham等，2020）。与全国互联网用户数交互是为了赋予截面工具变量以时间维度上的变异。两阶段模型设定如下：

$$DigEcon_{ct} = \pi_0 + \pi_1 \cdot Z_c \times InternetUsers_t + \delta \cdot Controls_{it} + \mu_i + \lambda_t + v_{it} \quad (2)$$

$$Innovation_{it} = \alpha + \beta \cdot \widehat{DigEcon}_{ct} + \gamma \cdot Controls_{it} + \mu_i + \lambda_t + \varepsilon_{it} \quad (3)$$

**策略二："宽带中国"试点政策的准自然实验**。2014年至2016年，国务院分三批批复了共117个"宽带中国"示范城市（城市群）。该政策显著推动了试点城市的数字基础设施建设和数字经济发展。本文以此构建双重差分模型：

$$Innovation_{it} = \alpha + \beta \cdot Treat_c \times Post_{ct} + \gamma \cdot Controls_{it} + \mu_i + \lambda_t + \varepsilon_{it} \quad (4)$$

其中，$Treat_c$为试点城市虚拟变量，$Post_{ct}$为政策实施时间虚拟变量。该模型配合平行趋势检验和安慰剂检验以验证DID估计的有效性。

### 3. 机制检验模型

参考江艇（2022）对因果推断中机制分析的方法论建议，本文采用以下模型检验知识溢出和资源配置效率两条传导渠道：

$$M_{it} = \alpha + \beta \cdot DigEcon_{ct} + \gamma \cdot Controls_{it} + \mu_i + \lambda_t + \varepsilon_{it} \quad (5)$$

其中$M_{it}$分别代入知识溢出代理变量和资源配置效率代理变量。本文重点检验数字经济对机制变量的因果效应（即X→M的关系），并结合理论分析论证机制变量对创新质量的作用逻辑（即M→Y的关系），而非简单套用传统的逐步回归中介效应检验方法。

### 4. 补充说明

**关于被解释变量的函数形式**。基准回归采用ln(专利数+1)作为被解释变量。考虑到专利数据属于非负计数变量且可能存在零膨胀问题（部分企业在部分年份的专利申请量为0），本文在稳健性检验中还采用泊松伪极大似然估计（PPML）进行重新估计（Correia等，2020），以检验结论对函数形式设定的敏感性。

**关于控制变量的选择**。控制变量中的研发投入强度（RD）可能同时受到数字经济的影响，存在"坏控制变量"的潜在问题。为此，本文在稳健性检验中报告了剔除RD后的回归结果，核心结论保持不变。此外，本文还以"发明专利占总专利申请量的比重"作为创新质量的替代指标进行了补充检验。
