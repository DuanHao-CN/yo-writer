# 一、引言

数字经济已成为重塑全球经济格局的核心力量。根据中国信息通信研究院发布的数据，2023年中国数字经济规模超过53万亿元，占GDP比重超过42%，数字经济增速显著高于同期GDP增速。从国家战略层面来看，"十四五"规划明确提出要加快数字化发展、建设数字中国，党的二十大报告进一步强调要加快发展数字经济，促进数字经济和实体经济深度融合。在此背景下，一个关键的学术问题是：数字经济的快速发展究竟在多大程度上推动了企业创新，尤其是推动了高质量的创新？

近年来，学术界围绕数字经济与企业创新的关系展开了广泛研究。已有文献从融资约束缓解、信息不对称降低、资源配置效率提升等多个角度，揭示了数字经济对企业创新活动的积极影响（赵涛等，2020；谢绚丽等，2018；唐松等，2020；吴非等，2021），为理解数字经济的创新驱动效应奠定了重要基础。

然而，现有文献在以下方面仍有拓展空间，这也构成了本文的研究切入点。**第一**，在创新测度方面，绝大多数研究以专利申请总量或研发投入作为企业创新的衡量指标，较少关注创新的"质量"维度。事实上，企业的创新活动在质量上存在显著差异：发明专利需经过严格的实质性审查，代表着真正的技术突破，属于"实质性创新"；而实用新型专利仅需形式审查，技术门槛较低，部分企业可能出于获取政府补贴、税收优惠或提升市场形象等策略性动机大量申请此类专利，形成所谓的"策略性创新"（黎文靖和郑曼妮，2016）。仅以专利总量评价创新水平，无法区分数字经济究竟是促进了实质性的技术进步，还是仅仅刺激了低质量的专利堆积。尽管已有少数研究关注了数字经济与专利结构的关系，但尚缺乏在严格因果推断框架下对数字经济如何差异化影响不同质量层次创新的系统检验。**第二**，在因果识别方面，部分研究在处理内生性问题上仍不够充分。数字经济发展水平较高的城市往往本身就具有更好的创新基础和制度环境，导致反向因果和遗漏变量偏误。本文针对这一问题，同时采用工具变量法（以1984年历史邮电局数量构建）和"宽带中国"试点政策的准自然实验，辅以PSM-DID、安慰剂检验等多重稳健性策略。**第三**，在作用机制方面，现有研究多停留在简单的中介效应检验层面。本文遵循江艇（2022）的方法论建议，从知识溢出和资源配置效率两条渠道分析数字经济影响创新"质量"的内在机制，并进一步从产权性质、企业规模、行业属性和区域发展水平四个维度考察异质性表现。

具体而言，本文利用2011—2022年中国A股上市公司面板数据，以北京大学数字普惠金融指数衡量城市数字经济发展水平，将企业创新产出区分为实质性创新（发明专利）和策略性创新（实用新型专利），系统考察数字经济对不同质量层次创新的差异化影响及其传导机制。

本文的边际贡献主要体现在以下三个方面。**第一**，本文拓展了数字经济与企业创新关系的研究视角。虽然已有学者从不同角度区分了创新质量（黎文靖和郑曼妮，2016；张杰等，2016），但鲜有研究在数字经济的情境下，结合严格的因果推断框架，系统考察数字经济对实质性创新与策略性创新的差异化影响。本文的研究发现——数字经济选择性地促进实质性创新——为这一领域提供了新的经验证据和分析视角。**第二**，本文提供了更加可靠的因果推断证据。通过同时采用工具变量法和准自然实验设计两种互补的识别策略，并辅以多项稳健性检验，本文有效缓解了内生性问题对估计结果的干扰。**第三**，本文揭示了数字经济影响企业创新质量的传导机制和异质性特征。知识溢出与资源配置效率两条渠道的识别不仅回答了"是否促进"的问题，还阐明了"如何促进"的逻辑；异质性分析揭示的结构性差异则为差异化的政策设计提供了经验依据。

本文后续安排如下：第二部分梳理相关文献；第三部分进行理论分析与研究假设推导；第四部分介绍研究设计；第五部分报告实证结果；第六部分进行机制检验；第七部分开展异质性分析；第八部分为结论与政策启示。
