# Paper Metadata

## Title Candidates
1. 数字经济发展与企业创新质量——基于实质性创新与策略性创新的视角
2. 数字经济如何影响企业创新质量？——来自中国上市公司专利数据的证据
3. "真创新"还是"凑数量"？数字经济对企业创新质量的影响研究

## One-Line Selling Point
本文从创新质量维度揭示了数字经济选择性地促进企业实质性创新而非策略性创新的因果效应及其内在机制，为理解数字经济的创新驱动作用提供了更精细的证据。

## Contribution List
1. 区分实质性创新（发明专利）与策略性创新（实用新型专利），揭示数字经济对企业创新"质量"而非仅"数量"的差异化影响，深化了数字经济与企业创新关系的研究。
2. 利用工具变量法与"宽带中国"试点政策的准自然实验，系统解决了数字经济与企业创新之间的内生性问题，提供了更可靠的因果推断证据。
3. 从知识溢出与资源配置效率两条渠道揭示了数字经济促进企业实质性创新的作用机制，并进一步发现该效应在民营企业、中小企业、高科技行业和中西部地区更为显著，丰富了数字经济影响创新的异质性认识。

## Story Line Logic Chain
```
数字经济快速发展，但其对企业创新的影响是否只是"量"的增加？
→ 现有研究聚焦创新数量，忽略了创新质量（实质性 vs 策略性）
→ 我们的核心洞察：数字经济通过知识溢出和资源优化，降低"真创新"的门槛
→ 实证设计：双向FE + IV + "宽带中国"DID，区分发明专利与实用新型专利
→ 核心发现：数字经济显著促进发明专利，但对实用新型专利影响不显著
→ 政策启示：数字经济是驱动高质量创新的关键力量，应重点支持
```

## Key Hypotheses
- H1: 数字经济发展显著促进企业实质性创新（发明专利）
- H1a: 数字经济发展对企业策略性创新（实用新型专利）的影响不显著或显著更弱
- H2: 数字经济通过知识溢出渠道促进企业实质性创新
- H3: 数字经济通过提升资源配置效率促进企业实质性创新
- H4a: 该效应在民营企业中更为显著（vs 国有企业）
- H4b: 该效应在中小企业中更为显著（vs 大型企业）
- H4c: 该效应在高科技行业中更为显著（vs 传统行业）
- H4d: 该效应在中西部地区更为显著（vs 东部地区）
